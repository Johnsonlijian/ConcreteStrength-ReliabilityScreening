# Exclusions and Public Boundary

This public package intentionally excludes:

- raw third-party data;
- active manuscript drafts;
- cover letters;
- reviewer-response drafts;
- private author information;
- funding records;
- credentials;
- logs that may contain local paths or private context.

The included outputs are selected derived tables and figures intended to support reproducibility and review. Users should obtain raw public datasets from their original repositories.
